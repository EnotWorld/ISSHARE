Разработка онлайн-магазина

Проект по созданию полнофункционального онлайн-магазина для продажи товаров различных категорий. Включает функции добавления товаров в корзину, оформления заказа и оплаты.

Мобильное приложение для учета финансов

Проект по разработке мобильного приложения для отслеживания и учета личных финансов. Приложение позволяет пользователям создавать бюджеты, управлять расходами и доходами, а также анализировать свои финансовые показатели.
